# Repositorio de prueba para TFM OSINT

Este repositorio fue creado como parte del caso práctico del Trabajo Fin de Máster titulado:

**OSINT: Estudio de técnicas, herramientas y OSINT Framework**

La finalidad de este repositorio es servir como entorno controlado para demostrar cómo los repositorios públicos pueden formar parte de un análisis OSINT defensivo.

La información incluida en este repositorio es ficticia y se utiliza únicamente con fines académicos. No contiene credenciales reales, claves API reales, contraseñas reales ni información sensible de terceros.

## Objetivo

Mostrar cómo una fuente pública como GitHub puede contener información técnica útil para un análisis OSINT, siempre dentro de un enfoque ético, defensivo y no intrusivo.

## Contenido del repositorio

- `README.md`: descripción general del repositorio.
- `configuracion_ejemplo.txt`: archivo ficticio de configuración.
- `documentacion_prueba.md`: documentación técnica simulada.

## Aviso

Este repositorio no representa un sistema real. Todo el contenido es de prueba.
